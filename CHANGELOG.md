## 2.5

新功能：

- 「对话模型」/「文生图 API」支持配置多套方案，可随时切换不用重新配置
- 打开其他电子书格式不再转换为 `.txt`，而是转换为 Markdown（`.md`）
  - 电子书目录会映射为相应层级的 ATX `#` 标题
  - 内链/注脚：支持鼠标移上显示内容、跳转与回跳
  - 外链：点击打开系统默认浏览器
  - 底栏文件路径弹出菜单：新增「重新转换」选项
  - 以前转换的带有 `<<A>>`/`<<ID>>`/`<<IMG>>` 等特殊标记的 `.txt` 文件会被当成普通的 `纯文本` 文件，**请重新转换**
- 「编辑模式 → **AI 智能排版**」（需启用「AI 阅读助手」）
  - 可点击工具栏「AI 智能排版」一键全文排版，也可以在编辑器中选中文本后「右键 → AI 智能排版：选中文本」进行局部排版（长文建议分次排版）
  - 排版完成后会显示 **Diff 预览**，可对排版结果逐一确认，点「应用」后再一次性写回，「放弃」则主文档不变
    - **不同模型的排版结果可能差异较大**，_请自行测试_
  - 可在「设置 → 编辑」中自定义排版选项：
    - **清理 HTML 残留**：解码 `&#35498;` 与 `&nbsp;` 等 HTML 实体，并去除 `<br/>` 等 HTML 残留。
    - **修正硬换行**：将同一自然段内因排版产生的句中强行换行合并为一行。
    - **修正标点符号**：引号/括号配对、半角转全角、断句补标点等；不改动数字小数点与英文对白中的半角标点。
    - **统一对话符号**：将对话统一为 `“”` 或 `「」` 包裹。
    - **修正乱码**：尝试将 `锟斤拷` `�` 等乱码还原为合理汉字；_有一定误还原风险，请自行检查。_
    - **还原 * 屏蔽**：尝试根据上下文还原正文里的 `*` 和谐字；不处理整行分隔线（如 `***********`）；_可能误猜，请自行检查。_
    - **移除盗版水印**：删除句中插入的防盗版杂符（如 `月*漪〇/泣②sa/`）；_有一定误删风险，请自行检查。_
    - **移除广告/引流信息**：删除 `发布于 www.xxx.com` `看小说，就来 xxx 网` 等站宣水印；_有一定误删风险，请自行检查。_
    - **压缩空行**：自动应用「格式化：压缩空行」。
    - **行首缩进**：自动应用「格式化：行首缩进」。
  - 可通过「设置 → 技能 → 智能排版」自定义 AI 排版行为。

改进：

- 字体列表：支持将其他字体**固定**到外层列表
- 信息栏：编辑模式下显示光标位置和选中字符数

修复：

- 章节匹配规则变更，标题行会正确格式化
- 更新角色立绘，角色卡能正常同步
